# Frontend EcoMida

Este diretorio contem o app mobile/web em React Native com Expo.

## Requisitos

- Node.js 18+
- npm 9+

## Como executar

No diretorio `frontend`:

```bash
npm install
npm run start
```

## Scripts principais

```bash
npm run start    # inicia o Expo
npm run android  # abre no Android
npm run ios      # abre no iOS
npm run web      # abre no navegador
npm run lint     # executa lint
```

## Reset do projeto

```bash
npm run reset-project
```
